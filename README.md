# xyclaw

OKX Windows 桌面量化交易程序。

**版本号**：`v1.0.2`  
**发布日期**：`2026-04-15`

## 功能概览
- 回测系统
- 真实仓位轮询与下单
- K 线图 / 总图 / 交易明细
- 本地历史数据缓存
- Windows 桌面打包产物

## 主要入口
- `app/desktop_app.py`
- `app/desktop_main.pyw`
- `app/services.py`
- `live/`
