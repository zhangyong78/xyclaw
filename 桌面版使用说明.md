# OKX 桌面版使用说明

## 1. 启动方式

直接双击下面这个文件：

- [启动OKX桌面程序.bat](/D:/xyclaw/启动OKX桌面程序.bat)

如果你想看报错窗口，用这个：

- [启动OKX桌面程序_调试.bat](/D:/xyclaw/启动OKX桌面程序_调试.bat)

## 2. 当前内置策略

- 只做多
- 周期：`1小时`
- 参数：`EMA21` 和 `EMA55`
- 开仓：`EMA21` 上穿 `EMA55` 后做多
- 止损：收盘价跌破 `EMA55` 后止损
- 平仓：`EMA21` 下穿 `EMA55` 后平仓

说明：

- 为了避免未来函数，信号按已收盘 K 线判断
- 回测和实盘共用同一套策略逻辑

## 3. 回测怎么用

在桌面程序左侧：

- 交易对填 `BTC-USDT`
- `1H 数据根数` 填要拉取多少根 1 小时 K 线
- `风险金` 填每笔最大风险金额
- 默认周期填 `1H`
- 点 `开始回测`

回测完成后：

- 下方表格会显示交易数、收益、回撤、得分
- 点 `打开报告` 可以查看 HTML 报告

## 4. 实盘/模拟盘怎么用

在桌面程序右侧：

- 不勾选 `允许真实下单`：只做信号检查，不会下单
- 勾选 `使用 OKX 模拟盘`：走 OKX 模拟盘
- 勾选 `允许真实下单`：满足条件时会向 OKX 发单

如果你不想连接真实账户：

- 直接手填 `计价币余额` 和 `基础币余额`

如果你要连接 OKX：

- 填 `API Key`
- 填 `API Secret`
- 填 `Passphrase`

然后：

- 点 `检查一次`：手动检查一次
- 点 `开始轮询`：按你设置的秒数持续检查
- 点 `停止轮询`：停止自动检查

## 5. 注意事项

- 第一次请先用 `模拟盘`
- 第一次请先不要勾选 `允许真实下单`
- 如果已经勾选真实下单，程序会先弹确认框
- 运行状态和订单结果会写到界面右下角日志区
